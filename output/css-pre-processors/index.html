  • LESS

    Less is a CSS pre-processor, meaning that it extends the CSS language, adding features that allow variables, mixins, functions and many other techniques that allow you to make CSS that is more maintainable, themable and extendable.

    Less runs inside Node, in the browser and inside Rhino. There are also many 3rd party tools that allow you to compile your files and watch for changes.

    Articles
    • Using the LESS CSS Preprocessor for Smarter Style Sheets

  • Stylus

    Expressive, dynamic, robust CSS. Installing Stylus is very easy once you have Node.js. So get the binaries for your platform and make sure that they also include npm, Node’s package manager.

    Articles
    • Compiling CSS using Stylus – Getting Started

  • Myth

    Myth is a preprocessor that lets you write pure CSS without having to worry about slow browser support, or even slow spec approval. It's like a CSS polyfill.
